### An online election platform by reactor

Try at [https://pwa-online-hackathon-2017.firebaseapp.com/](https://pwa-online-hackathon-2017.firebaseapp.com/)

##### What our application does
1. Allow each user to see a list of candidate and vote once.
2. Report bad behavior on any candidate.
3. View election result.

##### How we use PWA
1. Send notification of the election result
2. Allow app to works offline
3. Allow app to be added to home screen

##### How we use Firebase
1. Store all data inside Firebase Database
2. Store all images inside Firebase Storage
3. Use Firebase Hosting to host our website
